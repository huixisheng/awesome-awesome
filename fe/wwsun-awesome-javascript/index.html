# Awesome JavaScript

JavaScript是一个仅在一周就被发明出来的小玩意， 所以，它有着有多的缺陷，即使是这样，它已经存在了20年，全世界几乎每台电脑都至少装有一个JavaScript解释器， 况且今天这个当初的小玩意的目标是编写大型的企业应用。如果你对JavaScript的理解仅限于DOM操纵，页面动画，jQuery等等， 本列表将为你展现JavaScript的更多的令人震撼的功能。
